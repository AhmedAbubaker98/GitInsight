# Code of Conduct

## Our Commitment

We are committed to providing a welcoming, respectful, and harassment-free community for everyone.

Participation in this project means agreeing to this Code of Conduct in all project spaces.

## Expected Behavior

- Be respectful and constructive in communication.
- Give and receive feedback professionally.
- Assume good intent and ask clarifying questions.
- Focus discussions on technical substance and project goals.
- Respect different experience levels and backgrounds.

## Unacceptable Behavior

- Harassment, intimidation, or discrimination of any kind.
- Personal attacks, insults, or derogatory language.
- Sexualized language or unwelcome sexual attention.
- Trolling, inflammatory comments, or sustained disruption.
- Publishing private information without explicit permission.
- Any conduct that is inappropriate in a professional setting.

## Scope

This policy applies to project spaces including:

- GitHub issues, pull requests, and discussions
- Review comments and commit discussions
- Any official communication channels related to this project

## Enforcement

Maintainers are responsible for clarifying and enforcing this Code of Conduct.

Possible enforcement actions include:

1. Warning and request to change behavior
2. Temporary suspension from project spaces
3. Permanent ban from project participation

Severity and pattern of behavior determine the action taken.

## Reporting

If you experience or witness behavior that violates this Code of Conduct, report it privately to the maintainers through the repository owner profile contact.

When possible, include:

- Links or screenshots
- Date and time
- Description of what happened
- Any relevant context

All reports will be reviewed promptly and handled with discretion.

## Enforcement Principles

- Reports are handled as confidentially as possible.
- Retaliation against reporters is not tolerated.
- Maintainers may remove or edit content that violates this policy.
